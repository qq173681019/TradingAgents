# 📚 VSCode + MiniMax Agent 完整使用指南

## 🎯 核心理念

### 为什么选择 Web Agent 而不是 API？

✅ **CodingPlan 套餐优势**
- 固定订阅制，不按Token计费
- 包含足够的prompts额度
- 官方稳定的MiniMax M2模型支持
- 完整的Agent功能（不仅仅是API调用）

✅ **Web界面优势**
- 无需复杂的扩展配置
- 直接享受CodingPlan权益
- 更丰富的交互体验
- 支持图片、文件等多媒体输入

### 2. 分屏设置详解

#### 方案A：VSCode内嵌浏览器
```
1. 安装 "Simple Browser" 扩展
2. Ctrl+Shift+P → "Simple Browser: Show"
3. 输入: https://agent.minimax.io
4. 调整面板大小 (拖拽分隔线)
```

#### 方案B：外部浏览器分屏
```
1. Windows: Win+Left 让VSCode占左半屏
2. 浏览器占右半屏
3. 或使用超宽显示器的多窗口布局
```

#### 方案C：双显示器设置
```
主屏: VSCode编辑器
副屏: MiniMax Agent浏览器
```

## 🎨 界面定制

### VSCode设置优化

```json
// settings.json - 分屏优化
{
    "workbench.editor.splitSizing": "split",
    "workbench.panel.defaultLocation": "right",
    "workbench.editor.limit.enabled": true,
    "workbench.editor.limit.value": 8,
    "editor.minimap.enabled": true,
    "editor.minimap.side": "left"
}
```
