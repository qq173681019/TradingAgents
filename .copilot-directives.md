## 项目概述
- **项目名称**: 股票推荐与分析系统
- **技术栈**: Python 3.13

## 文件结构
TradingAgent/
├── TradingAgent/       # 业务逻辑
├── TradingShared/   # API接口及Key
    ├── config.py/   # 所有的API Key
    ├── path_config.py/   # 路径配置
├── StockMonitor/       # 股票自动交易系统，它会使用shared文件夹里的API
├── tools/       # 这些都是开发/测试工具，用于数据清理和功能验证


## 项目特定规则
1.Choice数据接口不用每次都尝试调用，除非我勾选了使用choice数据
2.如果勾选了使用choice，则所有评分和数据生成都会使用choice接口提供的数据
3.永远不要使用假数据
4.在做有关单只股票分析和筹码的计算时需要重新获取数据而不是用本地缓存，批量CSV分析时也如此
5.在做全部股票分数计算和生成推荐时，永远使用缓存数据，如果没有就给提示。
