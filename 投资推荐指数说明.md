## 📊 投资推荐指数体系

### 1. 综合推荐指数 (概览页面)
- **评分范围**: 0-100分
- **星级评价**: ★☆☆☆☆ 到 ★★★★★
- **可视化进度条**: 直观显示推荐程度
- **智能评级**: 强烈推荐、推荐、中性、谨慎、不推荐

#### 指数构成要素:
- 🏭 **行业因子**: 根据不同行业景气度调整
  - 半导体: +5至+15分 (政策支持)
  - 新能源: +8至+12分 (长期趋势)
  - 白酒: +3至+8分 (消费复苏)
  - 银行: 0至+5分 (稳定行业)
  - 房地产: -10至+5分 (政策敏感)

- 🏛️ **板块因子**: 不同板块特征加权
  - 科创板(688): +5至+10分 (创新加成)
  - 创业板(300): +3至+8分 (成长加成)
  - 沪市主板(60): +2至+6分 (稳定加成)
  - 深市主板(00): +1至+7分 (均衡加成)

### 2. 技术面推荐指数 (技术分析页面)
- **评分范围**: 0-100分
- **核心指标权重**:
  - RSI指标: ±15分 (超买超卖分析)
  - MACD信号: ±15分 (金叉死叉判断)
  - 成交量: ±10分 (资金关注度)
  - 均线排列: ±20分 (趋势强度)

#### 技术面评级标准:
- **80-100分**: 技术面强势 → 买入信号
- **65-79分**: 技术面偏强 → 可考虑买入
- **50-64分**: 技术面中性 → 持有观望
- **35-49分**: 技术面偏弱 → 谨慎操作
- **0-34分**: 技术面疲弱 → 回避风险

### 3. 基本面推荐指数 (基本面分析页面)
- **评分范围**: 0-100分
- **财务指标权重**:
  - PE估值: ±20分 (估值合理性)
  - ROE水平: ±20分 (盈利能力)
  - 营收增长: ±15分 (成长性)
  - 利润增长: ±15分 (盈利质量)

#### 基本面评级标准:
- **80-100分**: 基本面优秀 → 高质量公司
- **65-79分**: 基本面良好 → 质地较好
- **50-64分**: 基本面一般 → 中等质地
- **35-49分**: 基本面偏弱 → 质地偏弱
- **0-34分**: 基本面较差 → 需谨慎

### 4. 综合投资推荐指数 (投资建议页面)
- **权重分配**:
  - 技术面: 40%权重
  - 基本面: 60%权重
  - 市场环境: 动态调整
  - 行业景气: 动态调整

#### 综合评级体系:
- **85-100分**: ★★★★★ 强烈推荐 → 优质投资标的
- **75-84分**: ★★★★☆ 推荐 → 值得关注
- **65-74分**: ★★★☆☆ 中性 → 可适度配置
- **50-64分**: ★★☆☆☆ 谨慎 → 谨慎操作
- **0-49分**: ★☆☆☆☆ 不推荐 → 建议回避

## 🎯 如何使用投资推荐指数

### 1. 快速筛选
- 查看**综合推荐指数**，快速判断投资价值
- 85分以上：重点关注的优质标的
- 75-84分：可考虑配置的良好标的
- 65-74分：中性观望，等待更好时机
- 50-64分：谨慎操作，控制仓位
- 50分以下：建议回避

### 2. 深度分析
- **技术面指数**：判断买卖时机
  - 80分以上：技术面支持买入
  - 50-79分：技术面中性，看基本面
  - 50分以下：技术面偏弱，谨慎

- **基本面指数**：评估公司质地
  - 80分以上：高质量公司，长期持有
  - 65-79分：质地较好，可中长期配置
  - 50-64分：一般质地，短期操作
  - 50分以下：质地偏弱，建议回避

### 3. 组合配置
- **核心持仓** (50-70%): 综合指数85分以上
- **卫星持仓** (20-30%): 综合指数75-84分
- **机会型持仓** (10-20%): 综合指数65-74分，等待技术面改善

## 📈 实际操作建议

### 买入信号组合:
1. **综合推荐指数** ≥ 75分
2. **技术面指数** ≥ 65分
3. **基本面指数** ≥ 70分

### 持有观望组合:
1. **综合推荐指数** 65-74分
2. 等待技术面指数改善至65分以上

### 卖出信号组合:
1. **综合推荐指数** < 50分
2. **技术面指数** < 35分
3. **基本面指数**出现明显恶化

## ⚠️ 使用注意事项

1. **指数仅供参考**：投资推荐指数基于量化模型，不能替代独立判断
2. **动态调整**：指数会根据市场环境和数据变化实时调整
3. **风险控制**：无论指数多高，都要控制单一股票仓位
4. **综合考量**：结合自身风险承受能力和投资目标
