# A股智能分析系统 - 文件说明

## 🗂️ 项目文件结构

### 🚀 核心程序文件
- `a_share_gui_compatible.py` - 主GUI程序 (支持自动切换到命令行版本)
- `cli_launcher.py` - 功能完整的命令行版本
- `fix_launcher.py` - 应急启动器 (解决tkinter问题)
- `启动系统.bat` - Windows双击启动器

### 📊 数据文件
- `comprehensive_stock_data.json` - 完整的三时期股票分析数据
- `batch_stock_scores.json` - 批量评分数据缓存
- `stock_analysis_cache.json` - 单股分析缓存
- `sample_stocks_for_csv_analysis.csv` - 示例股票数据

### 📚 配置和文档
- `requirements.txt` - Python依赖包列表
- `README.md` - 项目主说明文档
- `GUI使用指南.md` - GUI版本使用指南
- `打包说明.md` - 程序打包说明
- `LICENSE` - 项目许可证

### ⚙️ 环境配置
- `.env.example` - 环境变量示例文件
- `.gitignore` - Git忽略文件配置

## 🎯 推荐使用方式

1. **最简单**: 双击 `启动系统.bat`
2. **命令行**: `python cli_launcher.py`
3. **应急启动**: `python fix_launcher.py`
4. **GUI尝试**: `python a_share_gui_compatible.py` (会自动切换)
