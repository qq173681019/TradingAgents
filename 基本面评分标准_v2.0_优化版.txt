│                                                                            │

┌──────────────────────────────────────────────────────────────────────────┐
│ 5. 现金流比率评分（权重8分）⭐ 新增指标                                 │
├──────────────────────────────────────────────────────────────────────────┤
│   现金流比率 = 经营性现金流净额 / 净利润                                │
│                                                                            │
│   比率 > 0.8:       +8分    ✅ 现金流充裕，盈利质量高                    │
│   0.5 < 比率 ≤ 0.8: +4分    ⚠️  现金流中等，需持续关注                  │
│   比率 ≤ 0.5:       -4分    ❌ 现金流不足，警惕财务造假                  │
│   默认值(无数据):    0分    不加减分                                     │
│                                                                            │
│   【新增理由】现金流是验证盈利真实性的关键指标                           │
│   【核心逻辑】利润可以做账，但现金流难以造假                             │
│   【优质标准】现金流/净利润 > 100%（现金奶牛）                           │
│   【风险信号】现金流长期为负（可能存在财务造假或应收账款堆积）           │
│   【适用场景】区分"真成长"和"假繁荣"                                    │
└──────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────┐
│ 6. 资产负债率评分（权重5分）⭐ 新增指标 + 行业差异化处理                │
├──────────────────────────────────────────────────────────────────────────┤
│   资产负债率 = 负债总额 / 资产总额                                      │
│                                                                            │
│   【非金融行业评分标准】                                                 │
│   负债率 < 30%:     +5分    ✅ 财务风险低，抗风险能力强                  │
│   30% ≤ 负债率 < 60%: 0分   ⚠️  财务风险中等，行业标准水平              │
│   负债率 ≥ 60%:     -5分    ❌ 财务风险高，杠杆过大                      │
│                                                                            │
│   【金融行业评分标准】⭐ 差异化处理                                      │
│   负债率 < 70%:     +5分    ✅ 极低负债（罕见，优质）                    │
│   70% ≤ 负债率 < 90%: +2分  ✅ 正常范围（符合商业模式）                  │
│   负债率 ≥ 90%:     -3分    ⚠️  负债率过高（风险警示）                  │
│                                                                            │
│   【新增理由】财务风险是长期投资的重要考量                               │
│   【核心逻辑】低负债率提供安全边际，高负债率增加破产风险                │
│   【行业差异】⭐ 重要优化                                                │
│     • 金融行业（银行/保险/券商）：70-90%负债率是正常商业模式             │
│       - 银行通过吸收存款放贷，负债率自然高                               │
│       - 保险通过收取保费投资，负债率也较高                               │
│       - 评分标准已调整，避免误杀优质金融股                               │
│     • 制造业：负债率30-50%合理                                           │
│     • 科技互联网：负债率<30%为佳（轻资产运营）                           │
│     • 房地产：负债率60-80%常见（但需警惕政策风险）                       │
│   【风险信号】                                                           │
│     • 非金融行业：负债率>80%且持续上升（资金链紧张）                     │
│     • 金融行业：负债率>90%且不良率上升（资产质量恶化）                   │
│   默认值(无数据): 0分 不加减分                                           │
└──────────────────────────────────────────────────────────────────────────┘

================================================================================
                            计算公式详解
================================================================================

【原始分数计算 v2.0】
原始分数 = 50 (基础分)
         + PE评分 (-14 ~ +18)
         + ROE评分 (-9 ~ +18)
         + 营收增长评分 (-12 ~ +12)
         + 利润增长评分 (-12 ~ +12)
         + 现金流评分 (-4 ~ +8)      ⭐新增
         + 负债率评分 (-5 ~ +5)      ⭐新增
         + 毛利率评分 (0 ~ +3)       ⭐新增
         + 行业加成 (-2 ~ +7)        ⭐优化

【理论分数区间 v2.0】
最低分: 50 - 14 - 9 - 12 - 12 - 4 - 5 + 0 - 2 = -8分
最高分: 50 + 18 + 18 + 12 + 12 + 8 + 5 + 3 + 7 = 133分
实际区间: [-8, 133]

【最终评分转换】
最终评分 = min(10.0, max(1.0, 原始分数 / 10))
转换后范围: [1.0, 10.0]

【评分等级划分】
9.0 - 10.0 分:  ⭐⭐⭐⭐⭐ 优质资产，强烈推荐（顶级标的）
8.0 - 8.9 分:   ⭐⭐⭐⭐   优秀公司，值得关注（核心持仓）
7.0 - 7.9 分:   ⭐⭐⭐     中等偏上，可以考虑（卫星仓位）
6.0 - 6.9 分:   ⭐⭐       及格水平，谨慎参与（小仓试水）
5.0 - 5.9 分:   ⭐         一般水平，不建议（观望为主）
1.0 - 4.9 分:   ❌         基本面差，回避（坚决不碰）

   • 现金流和负债率作为安全边际指标

   • 周期性行业中性评分机制
   • 政策压制行业减分机制

4. 【权重分配科学】
   • 盈利质量（ROE 18分 + 现金流 8分）= 26分权重

   • 财务风险（负债率5分）守住安全底线

❗ 业绩预告下修风险：关注业绩预告和业绩快报

❗ 建议：

   ✅ 跟踪经营公告（重大合同、订单变化）

❗ 基本面优秀不代表股价一定上涨（市场情绪、流动性影响）
❗ 黑天鹅事件无法预测（财务造假、政策突变、行业变革）
❗ 需结合宏观经济环境和市场流动性
❗ 行业加成具有时效性，需动态调整（政策导向变化快）

❗ 科技股：研发投入高可能短期拖累利润，需长期视角

float (1.0 ~ 10.0)

                        关键优化详解（第二轮）
================================================================================

【优化1: 行业差异化处理 - 负债率评分】⭐ 已实施

问题：
  金融股（银行、保险、券商）商业模式决定负债率70-90%是正常范围
  如果用统一标准（负债率>60%扣5分），会误杀所有优质银行股

解决方案：
  代码层面：
    • 增加行业判断逻辑（识别银行、保险、券商、信托、金融关键词）
    • 金融股负债率阈值调整：<70% +5分，70-90% +2分，>90% -3分
    • 非金融股保持原标准：<30% +5分，30-60% 0分，>60% -5分
  
  文档层面：
    • 在负债率评分章节详细说明行业差异
    • 新增招商银行案例（负债率82%，评分9.9/10）
    • 对比说明：如不差异化处理会被扣5分（误杀）

实际效果：
  • 招商银行（负债率82%）：优化前可能7.0分 → 优化后9.9分 ✅
  • 普通制造业（负债率65%）：仍然-5分（风险警示）✅

【优化2: 数据时效性警示】⭐ 已实施

问题：
  基本面评分基于财报数据，存在1-3个月滞后（最长4个月）
  用户可能不了解数据时效性，导致投资决策失误

解决方案：
  文档层面：
    • 新增"数据时效性警示"专门章节
    • 明确说明财报发布时间（每季度结束后1个月内）
    • 列出数据滞后期影响（1-3个月，最长4个月）
  
  实用建议：
    • 建议配合业绩预告（提前1-2个月预警）
    • 关注重大经营公告（合同、订单、产能变化）
    • 跟踪行业动态和政策变化
    • 参考机构调研和券商研报
    • 结合高频数据（月度产销量、开工率）
  
  风险提示：
    • 业绩变脸风险：财报发布前可能业绩预告下修
    • 黑天鹅事件：突发事件导致基本面急剧恶化
    • 政策突变：行业监管政策短期内大幅调整

实际效果：
  • 用户了解数据局限性，不盲目依赖历史数据
  • 主动关注业绩预告和经营公告，降低风险
  • 定期复核基本面（每季度财报后），及时调整持仓
