# 🚀 A股智能分析系统 - 项目文件说明

## 📁 核心文件结构

### 🎯 **主要程序文件**
- **`a_share_gui_compatible.py`** - 主GUI应用程序，提供用户界面
- **`comprehensive_data_collector.py`** - 核心数据采集引擎，支持多数据源
- **`requirements.txt`** - Python依赖包列表
- **`启动系统.bat`** - Windows一键启动脚本

### 📊 **数据文件**
- **`data/comprehensive_stock_data.json`** - 完整股票数据存储
- **`data/pre_collected_stock_data.json`** - 预收集数据
- **`batch_stock_scores.json`** - 批量股票评分数据

### 📖 **文档文件**
- **`README.md`** - 项目主要说明文档
- **`GUI使用指南.md`** - GUI界面使用说明
- **`打包说明.md`** - 项目打包部署说明
- **`LICENSE`** - 开源许可证

### ⚙️ **配置文件**
- **`.gitignore`** - Git忽略文件配置
- **`.env.example`** - 环境变量配置示例

## 🚀 **启动方式**

### Windows用户
```bash
# 双击运行
启动系统.bat

# 或命令行运行
python a_share_gui_compatible.py
```

### 其他系统
```bash
# 安装依赖
pip install -r requirements.txt

# 启动程序
python a_share_gui_compatible.py
```

## ✨ **主要功能**

1. **🎯 智能数据采集**
   - 多数据源轮换：tushare ↔ akshare (K线数据)
   - 三级保障机制：腾讯 → yfinance → baostock (基础信息)
   - 批量优化：15只股票/批，最大化效率

2. **📊 综合数据支持**
   - K线数据、基础信息、财务数据
   - 行业概念、资金流向、技术指标
   - 新闻公告、智能评分分析

3. **🖥️ 用户友好界面**
   - 现代化GUI设计
   - 实时进度显示
   - 一键批量数据采集

4. **⚡ 性能优化**
   - 智能频率控制
   - 多层数据源保障
   - 高效批量处理
