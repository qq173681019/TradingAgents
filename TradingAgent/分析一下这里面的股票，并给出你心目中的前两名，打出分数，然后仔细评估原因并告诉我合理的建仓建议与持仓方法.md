# 推荐系统更新说明

## 更新日期
2026-03-15

## 更新内容

### 1. 推荐策略调整

#### 原推荐策略（已废弃）
- 综合推荐：技术35% + 基本20% + 筹码40% + 热门板块5%
- 基本推荐：技术10% + 基本45% + 筹码40% + 热门板块5%
- 筹码推荐：技术10% + 基本10% + 筹码70% + 热门板块10%
- 技术推荐：技术80% + 基本10% + 筹码10% + 热门板块0%

#### 新推荐策略

**短线推荐**（`推荐_短线.csv`）
- 技术面评分：40%
- 筹码健康度：40%
- 板块趋势评分：20%

**长线推荐**（`推荐_长线.csv`）
- 基本面评分：50%
- 筹码健康度：50%

**板块趋势分析**（`板块趋势分析.csv`）
- 各板块未来趋势评分
- AI板块前景分析报告

### 2. 板块趋势分析

新增板块趋势分析功能，基于以下因素：
- 板块内股票的平均技术面评分
- 板块内股票的平均基本面评分
- 板块内股票的平均筹码面评分

评分标准：
- 7分以上：板块表现强劲，未来趋势看好
- 6-7分：板块表现良好，未来趋势积极
- 5-6分：板块表现一般，需谨慎观察
- 5分以下：板块表现较弱，建议谨慎观望

### 3. 短线推荐特点

**主要关注：**
- 技术面评分（短期走势）
- 筹码健康度（短期资金流向）
- 板块趋势（是否处于上涨及利好区间）

**适用场景：**
- 短期交易
- 追涨杀跌
- 技术面分析

### 4. 长线推荐特点

**主要关注：**
- 基本面评分（公司业绩、成长性）
- 筹码健康度（长期筹码结构）

**适用场景：**
- 中长期投资
- 价值投资
- 基本面分析

## 使用方法

### 自动执行
双击运行「自动更新K线并生成主板推荐.bat」即可自动完成：
1. 更新K线数据
2. 整理主板基础评分数据
3. 生成短线和长线推荐
4. 导出板块趋势分析

### 导出文件
执行完成后，桌面会生成以下CSV文件：
1. `推荐_短线.csv` - 短线推荐股票（前10名）
2. `推荐_长线.csv` - 长线推荐股票（前10名）
3. `板块趋势分析.csv` - 各板块趋势分析

## 技术说明

### 评分体系
所有评分均为1-10分制：
- 10分：优秀
- 8-9分：良好
- 6-7分：一般
- 5分：中等
- 4分以下：较差

### 板块趋势分析
当前使用简化版分析，基于板块内股票的平均表现。

未来可扩展：
- 集成真正的AI大模型进行板块趋势预测
- 加入宏观经济因素分析
- 加入政策面分析
- 加入资金流向分析

### 数据来源
- K线数据：通过akshare等数据源获取
- 基本面数据：财务报表、业绩预告等
- 筹码数据：股东人数、持仓集中度等
- 板块数据：行业分类、热门板块等

## 注意事项

1. **仅供参考**：所有推荐仅供参考，不构成投资建议
2. **风险提示**：股市有风险，投资需谨慎
3. **及时更新**：建议每日更新K线数据，保证数据时效性
4. **综合判断**：建议结合多个指标综合判断，不依赖单一推荐
