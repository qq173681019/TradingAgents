## 🎯 立即开始使用

### 方法1: 一键启动 (推荐)
```bash
# Windows用户
双击运行: open-minimax-agent.bat

# Mac/Linux用户  
chmod +x open-minimax-agent.sh
./open-minimax-agent.sh
```

### 方法2: VSCode任务
```
1. 按 Ctrl+Shift+P 打开命令面板
2. 输入 "Tasks: Run Task"
3. 选择 "启动 MiniMax CodingPlan 开发环境"
```

### 方法3: 手动设置
```
1. 启动VSCode: code .
2. 按 Ctrl+\ 分屏
3. 右侧访问: https://agent.minimax.io
4. 开始AI辅助编程
```
