Web版提供了完整的REST API，可用于编程调用:

【1. 分析单只股票】
  GET /api/analyze/600519
  
  返回:
  {
    "success": true,
    "code": "600519",
    "name": "贵州茅台",
    "data": {
      "price": 1250.50,
      "scores": {
        "technical": 7.5,
        "fundamental": 8.2,
        "comprehensive": 7.9
      },
      ...更多数据...
    }
  }

【2. 批量评分】
  POST /api/batch-score
  Body: {"stocks": ["600519", "600036"], "use_llm": false}
  
  返回:
  {
    "success": true,
    "total": 2,
    "scored": 2,
    "results": {
      "600519": {...},
      "600036": {...}
    }
  }

【3. 获取推荐】
  GET /api/recommendations?min_score=6.0&type=all
  
  返回:
  {
    "success": true,
    "recommendations": [...]
  }

【4. 健康检查】
  GET /api/health
  
  返回:
  {
    "status": "online",
    "gui_ready": true,
    "timestamp": "..."
  }
